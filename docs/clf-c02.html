CLF-C02 Quick Reference

Key AWS AI services, concepts, and exam tips at a glance.

☁️ Cloud Concepts (24%)
6 AdvantagesCapEx→OpEx, Elasticity, Economies of Scale, Speed, Global, No DC
Shared ResponsibilityAWS: hardware/hypervisor/managed services. You: OS/apps/data/IAM
Well-ArchitectedOpEx, Security, Reliability, Perf Efficiency, Cost, Sustainability
IaaS/PaaS/SaaSEC2=IaaS, Beanstalk=PaaS, WorkMail=SaaS
Region vs AZRegion=geo area, AZ=isolated DC(s) in region, Edge=CDN PoP
Deployment ModelsPublic, Private, Hybrid, Community cloud
Migration 6 RsRehost, Replatform, Refactor, Repurchase, Retire, Retain
RTO vs RPORTO=max downtime, RPO=max data loss
🔒 Security & Compliance (30%)
IAMUsers, Groups, Roles (temp creds), Policies (JSON)
Least PrivilegeGrant only minimum permissions needed for the task
MFARequired for root account; TOTP or hardware key
KMS vs CloudHSMKMS=managed multi-tenant; CloudHSM=dedicated FIPS 140-2 L3
GuardDutyML threat detection on CloudTrail/VPC/DNS logs, no agents
WAF vs ShieldWAF=Layer7 web attacks; Shield=DDoS (Standard=free)
CloudTrailLogs ALL API calls; audit & compliance trail
AWS ArtifactOn-demand compliance reports (SOC, PCI, ISO, HIPAA BAA)
SG vs NACLSG=stateful instance-level; NACL=stateless subnet-level
SCPsOrg-wide permission guardrails; override even admin permissions
⚙️ Cloud Technology (34%)
EC2 PricingOn-Demand, Reserved (1/3yr), Spot (90% off, interruptible), Dedicated
S3 Storage ClassesStandard, Standard-IA, One Zone-IA, Glacier, Glacier Deep, Int-Tiering
RDS Multi-AZ vs RRMulti-AZ=HA failover; Read Replicas=scale reads
LambdaServerless, event-driven, max 15min, pay per invocation
SQS vs SNSSQS=pull queue (1:1); SNS=push pub/sub (1:many)
VPCPrivate network; subnets, route tables, IGW, NAT GW, SG, NACL
CloudWatchMetrics, logs, alarms, dashboards, events/EventBridge
CloudFormationInfrastructure as Code; JSON/YAML templates; free (pay resources)
ALB vs NLBALB=Layer7 path routing; NLB=Layer4 ultra-high performance
Route 53Managed DNS; Simple, Weighted, Latency, Failover, Geolocation
💰 Billing & Pricing (12%)
Free Tier TypesAlways Free, 12-Month Free (new accounts), Short-term Trials
Cost ExplorerVisualize & forecast spend; filter by service/tag/account
AWS BudgetsAlerts on actual & FORECASTED cost/usage thresholds
Savings PlansCompute (EC2+Fargate+Lambda) vs EC2 Instance (higher discount)
Pricing CalculatorEstimate costs BEFORE building; model architecture expenses
Data TransferInbound=free; Outbound to internet=charged per GB
OrganizationsConsolidated billing; volume discounts; single invoice
Trusted Advisor5 categories: Cost, Security, Reliability, Perf, Service Limits
TCOCompare total on-prem costs (power/cooling/staff) vs AWS
