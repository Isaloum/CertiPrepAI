🧠 The 40 Core Concepts You Must Memorize First

These appeared across both exams. Master these before taking another practice exam.

RDS Multi-AZ Failover
CNAME switches from primary to standby. IP does NOT move. New instance is NOT created.
RDS Multi-AZ vs Read Replica
Multi-AZ = HA, synchronous, automatic failover. Read Replica = async, for read scaling only.
Aurora Reader Endpoint
Load balances automatically across ALL read replicas. Use this instead of individual replica endpoints.
RDS Proxy
Connection pooling for Lambda→RDS. Fixes "too many connections" errors. Sits between Lambda and RDS.
Aurora native Lambda
lambda_sync / lambda_async — invoke Lambda directly from Aurora MySQL stored procedures.
DMS CDC
Full load + Change Data Capture = ongoing replication. SSL via CA certificate on DMS endpoint.
DynamoDB Gateway Endpoint
VPC → DynamoDB without internet. Routes via route table. NOT an interface endpoint. Use AWS Backup for cross-account copies.
ElastiCache Redis AUTH
Password-protect Redis with AUTH command. NOT IAM, NOT security groups alone.
S3 Static Website Hosting
Bucket name MUST match domain name. Enable website hosting. Point Route 53 Alias record to S3 endpoint.
S3 + CloudFront
OAI (Origin Access Identity) restricts S3 to CloudFront only. Use pre-signed URLs for time-limited access.
S3 Object Lock
WORM model. Compliance mode = no one can delete (incl. root). Governance = selected users can override.
S3 Standard-IA
Infrequent access but RAPID retrieval. Low latency, high throughput. NOT for archiving (that's Glacier).
S3 Encryption Headers
SSE-S3: x-amz-server-side-encryption. SSE-C uses 3 other headers (customer-key, algorithm, key-MD5).
AWS DataSync
Bulk on-prem→AWS data migration. Can target S3 Glacier Deep Archive directly. Faster than Storage Gateway for one-time moves.
Storage Gateway Tape Gateway
Replace physical tapes. Virtual tapes → S3 → S3 Glacier Deep Archive. Best for existing tape backup workflows.
Amazon DLM
Data Lifecycle Manager. Automates EBS snapshot creation + retention. No extra cost, no custom scripts needed.
FSx for Lustre
High-performance parallel file system for ML/HPC. NOT EFS. Handles hot/warm data at massive throughput.
CloudFront Lambda@Edge
Runs Lambda at edge locations. Use for: dynamic content based on User-Agent, A/B testing, auth. CF Functions = simpler/faster, no network calls.
CloudWatch Metrics
Built-in EC2: CPU, Network, Disk I/O. NOT Memory, NOT Swap. Install CloudWatch Agent for custom metrics.
Route 53 Routing Policies
Geolocation = fixed by country. Geoproximity = bias to expand/shrink coverage. Latency = lowest latency. Weighted = % split.
Route 53 Failover
Active-passive with health checks. Healthy primary → DNS points to primary. Primary down → DNS points to secondary (e.g., S3 static site).
ALB vs NLB vs GLB
ALB = L7 (HTTP/HTTPS/gRPC, path/host routing). NLB = L4 (TCP/UDP, EIP support). GLB = L3/L4 (GENEVE, 3rd-party appliances).
EIP on Load Balancer
EIP can ONLY be attached to NLB, NOT ALB. ALB uses DNS names only. Use NLB + EIP when clients whitelist IP addresses.
ENI Failover
Secondary ENI retains private IP/MAC. Detach from failed instance, attach to standby → traffic resumes. No DNS change needed.
VPC Peering — No Transitive Routing
VPC-A peers VPC-B. VPC-B peers VPC-C. VPC-A CANNOT reach VPC-C via VPC-B. No edge-to-edge routing.
Egress-Only IGW
IPv6 outbound only from private subnet. Like NAT Gateway but for IPv6. Cannot be used for IPv4.
AWS Network Firewall
Stateful L3-L7 inspection for VPC. Blocks malicious URLs and FQDNs. ALB+WAF = L7. Network Firewall = deeper VPC-level filtering.
WAF
L7 rules: SQL injection, XSS, rate-based, geo-match. Attaches to: ALB, CloudFront, API Gateway. NOT NLB.
Auto Scaling Cooldown
Default: 300 seconds. Prevents ASG from launching/terminating before previous activity takes effect. Does NOT mean instances "cool down."
IAM Role on EC2
ALWAYS use IAM roles for EC2 to access AWS services. NEVER store access keys on the instance. MFA for extra admin protection.
AWS STS
Issues temporary credentials via AssumeRole. Cross-account access. NOT Cognito (that's for app users). NOT IAM Identity Center (that's SSO).
AWS Organizations + Control Tower
Organizations = multi-account management + consolidated billing. Control Tower = automated landing zone with guardrails. IAM Identity Center = SSO with directory.
SNS Filter Policies
One SNS topic → multiple SQS queues with filter policies. Each queue only receives matching messages. Fan-out pattern.
SQS Priority Queue Pattern
No built-in priority in SQS. Solution: two separate queues (premium + free). EC2 polls premium first, then free if empty.
AWS Glue for ETL
Crawler discovers schema in S3. ETL job converts CSV→Parquet. Scheduled or on-demand. Fully serverless.
EMR + Redshift
"Big data processing frameworks" = EMR. "Standard SQL + BI tools" = Redshift. EMR→Redshift is the analytics pipeline.
Secrets Manager
Stores and ROTATES secrets automatically. SSM Parameter Store does NOT auto-rotate. KMS = encryption keys only.
GuardDuty
Threat detection using ML/anomaly detection. Detects suspicious S3 access, malicious IPs. Config = compliance rules. CloudTrail = API audit log.
AWS Config
Continuous audit of resource configurations. Create rules to detect non-compliance (e.g., public S3 buckets). Config ≠ CloudTrail.
Snowball vs DataSync vs Direct Connect
Snowball/Edge = large one-time physical migration (TB-PB). DataSync = online transfer (fast, but uses network). Direct Connect = permanent dedicated line.
Amazon Rekognition
Content moderation, face detection, object recognition. Interface VPC endpoint for private access. Kinesis Video Streams feeds live video to Rekognition.
🆕 Exam 3 — New Concepts That Also Failed

These appeared in Exam 3 and are in addition to the 40 concepts from Exams 1 & 2. Many are SAA-C03 basics.

SSH = Port 22, RDP = Port 3389
Allow TCP 22 inbound in Security Group to enable SSH. /32 = single IP, /0 = all IPs.
S3 Lifecycle Policy
Automatically transition objects to cheaper storage (Glacier) or delete them after N days. No EC2, no Lambda needed.
IAM Groups
Create group per department. Attach policy to group. Add users to group. NEVER attach policies to individual users at scale.
Spot Instances
Cheapest option. Use for interruptible, fault-tolerant workloads (batch, video transcoding, HPC). Can be interrupted with 2-min notice.
EC2 Stop/Start Behavior
Instance store data = LOST on stop. EBS = persists. Host may change. EIP stays associated (in VPC). ENI stays attached.
Amazon Data Firehose
Loads streaming data to S3, Redshift, OpenSearch, Splunk. Fully managed. Use for near-real-time analytics. NOT for reading/processing — that's Kinesis Data Streams + Lambda.
Direct Connect + Transit Gateway
One DX + Transit Gateway = connect multiple VPCs/accounts. Avoid setting up separate DX per account (expensive). VPC peering doesn't support edge-to-edge.
Kinesis Data Streams — Retention
Default: 24 hours. Max: 365 days. If consumer processes every other day → data expires. Increase retention period or process more frequently.
S3 + Athena for Log Analysis
ELB/ALB access logs → S3. Analyze with Athena (serverless SQL). No EC2, no custom app needed. Cheapest and simplest.
EC2 vCPU Limit
EC2 limits are per REGION (not per AZ). Submit limit increase form via Service Quotas. You don't need to change regions.
NACL Ephemeral Ports
NACLs are stateless. Inbound 443 allowed → also need outbound 32768-65535 for return traffic. Security Groups are stateful (auto-allow return).
EBS DeleteOnTermination
Default: root volume deleted on instance termination. Set DeleteOnTermination=false to preserve root volume. Set at launch or via CLI.
EC2 Cluster Placement Group
Packs instances close in ONE AZ for low-latency, high-throughput node-to-node communication. Required for HPC. Cannot span regions.
EC2 Hibernation
Must be enabled at LAUNCH. Cannot enable after. Saves RAM to EBS. Faster restart. Pay only for EBS + EIP while hibernated.
VPC Subnet Facts
Each subnet = 1 AZ only (cannot span). New subnets linked to main route table by default. VPC CIDR: /16 (65536 IPs) to /28 (16 IPs).
AWS Glue Job Bookmark
Tracks last processed data. Next run only processes NEW data. Prevents reprocessing old data. Enable in ETL job settings.
S3 Object Lambda
Transform S3 data on GET request. Lambda receives bucket name + object KEY. Use key (filename) to identify and process correct object.
RDS Read Replica
Async replication. Read-only. Scales read traffic. Multiple replicas = scale further. Multi-AZ = sync, for HA/failover only.
SQS + Auto Scaling
Scale EC2 based on ApproximateNumberOfMessages. Backlog per instance = queue depth / running instances. Target value = max latency / processing time.
Lambda KMS Decrypt
Add kms:decrypt to Lambda EXECUTION ROLE (not resource policy). Also update KMS key policy to grant the execution role ARN permission.
Lambda Container Image
Fully serverless. Can be up to 10 GB. Ephemeral storage: default 512MB, up to 10 GB configurable. Use EFS for persistent shared storage.
AWS Backup for RDS
RDS automated backup max = 35 days. For 90+ days: use AWS Backup plan. Supports cross-account, cross-region copies.
VPC Peering — Direct Only
No transitive routing. DEV↔UAT and UAT↔PROD does NOT mean DEV↔PROD. Must create direct DEV↔PROD peering connection.
AWS Config Tag Rule
required-tags managed rule checks for missing tags. Least effort for tag compliance. SCP = prevents actions, doesn't detect non-compliance.
RDS Storage Autoscaling
Enable autoscaling on RDS to expand storage automatically. Zero downtime. Least overhead vs manually increasing allocated storage.
EC2 Billing Edge Cases
Stopping to HIBERNATE = still billed. Stopping normally = not billed. Reserved Instance in terminated state = still billed until term ends.
SSM Parameter Store vs Secrets Manager
SSM SecureString = cheap, KMS encrypted, no auto-rotation. Secrets Manager = auto-rotation, more expensive ($0.40/secret/mo). Use SSM for static config, Secrets Manager for rotating credentials.
Storage Gateway File Gateway
On-prem SMB/NFS → S3. Supports local caching for low-latency access. Use for on-prem backup to cloud with existing protocols.
Glacier Expedited + Provisioned
Expedited = 1-5 min retrieval. Provisioned Capacity = guarantees 3 expedited retrievals per 5 min + 150 MB/s. Required for guaranteed SLA.
S3 Transfer Acceleration
Global uploads via CloudFront edge locations. Best for users uploading to a centralized S3 bucket from around the world. Not for downloads.
Reserved Instance Marketplace
Sell unused Standard Reserved Instances. Terminate instances to stop new charges. RIs still bill until term ends unless sold in Marketplace.
Auto Scaling Launch Template
Cannot modify existing launch template. Create a NEW launch template to use a new AMI. Target groups are for load balancers, not ASGs.
Lambda in VPC — ENI Exhaustion
Lambda needs ENIs + private IPs to connect to VPC. If subnet has limited IPs or VPC hits ENI quota → EC2ThrottledException. Use larger subnet CIDR.
S3 Event Types
ObjectCreated:* = all uploads. ObjectRemoved:Delete = permanent delete. ObjectRemoved:DeleteMarkerCreated = versioned object marker only.
Transcribe → Translate → Comprehend
Audio→Text: Amazon Transcribe. Text translation: Amazon Translate. Sentiment analysis: Amazon Comprehend. No ML training needed — all managed APIs.
ALB SNI — Multiple Certs
Upload all SSL certs to ALB. Bind multiple certs to same HTTPS listener. ALB uses SNI to pick the right cert per client. Free. No re-auth needed.
S3 Cross-Account Copy
Source bucket policy + destination IAM policy. Use IAM role in destination account to perform copy. Destination account owns the copied object.
CloudFormation CreationPolicy
Prevents stack from completing until cfn-signal sends success signal. Use with EC2 instances to wait for app install. DependsOn = just ordering, not success.
S3 Gateway Endpoint
FREE. Private subnet → S3 without internet/NAT. Add to route table. Interface endpoint costs hourly. NAT Gateway costs hourly + data. Gateway = cheapest.
DynamoDB + AppSync
Real-time, schemaless voting/social app = DynamoDB. AppSync provides real-time GraphQL API with subscriptions. Schema changes = no ALTER TABLE needed.
EKS + Fargate
Serverless Kubernetes. No EC2 node management. Cluster Autoscaler handles pod scaling. Each pod = isolated compute. Pay per pod resource.
NACL First-Match Rule
NACLs evaluate rules in order (lowest number first). First match wins. Rule 100 allows all → rule 101 deny never reached. Unlike Security Groups (all rules evaluated).
Route 53 Active-Active
All resources available simultaneously. Use weighted/latency/geolocation routing. Active-Passive = primary + standby. Active-Active = everyone serves traffic.
Client-Side Encryption — Two Options
1. Client-Side with KMS key (AWS manages key rotation). 2. Client-Side with client-side master key (you manage rotation). Both encrypt BEFORE sending to S3.
SCP Restricts AWS Config
SCPs set guardrails for entire OU. Even admins can't override SCP. Attach SCP to OU to block access to AWS Config for all accounts in it.
Cost Explorer API
Programmatic access to cost + usage data. Supports pagination for large datasets. Use for building automated cost reports. Budgets = alerts only.
